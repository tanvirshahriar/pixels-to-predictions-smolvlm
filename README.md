# Pixels to Predictions: SmolVLM QLoRA

This repository contains my final work for the Pixels to Predictions deep learning vision challenge.

## Files

- `final_notebook.ipynb`: Kaggle notebook for loading data, training the adapter, evaluating validation accuracy, and generating `submission.csv`.
- `submission.csv`: within-deadline Kaggle submission generated from the 1500-step adapter run.
- `adapter_1500_submitted/`: LoRA/DoRA adapter weights for the within-deadline submitted model.
- `adapter_post_deadline_best/`: adapter weights for the later full-data run that completed after the deadline due to a Kaggle crash.
- `report.tex`: LaTeX source for the final report.
- `report.pdf`: compiled final report.
- `requirements.txt`: package notes.

## Model

The base model is:

`HuggingFaceTB/SmolVLM-500M-Instruct`

The full base model weights are not included in this repository. The notebook expects the model to be available locally in the Kaggle input directory.

## Results

- Within-deadline submitted run: 1500-step adapter, public score 0.56.
- Post-deadline best run: full 3,109-sample run, validation accuracy 0.6100, public score 0.74446 when evaluated afterward.

## Reproducibility

The notebook uses only the provided competition data and images. It writes predictions to `submission.csv` with columns:

`id,answer`
